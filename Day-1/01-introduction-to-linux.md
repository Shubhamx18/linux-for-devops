## âš™ï¸ Linux is Not a Complete Operating System

Linux itself is only a **kernel**.

To become a usable operating system, it is combined with:

- GNU command-line utilities  
- A shell (Bash, Zsh, etc.)  
- System libraries  
- Package management tools  
- Optional graphical interfaces  

This complete system is called a **Linux Distribution**.

## ğŸ“¦ Linux Distributions

A **Linux Distribution (Distro)** packages the Linux kernel with essential tools and software.

| Distribution | Best Known For |
|--------------|----------------|

| **Fedora** | Latest technologies |
| **Rocky Linux / AlmaLinux** | Enterprise servers |

All distributions share the same kernel but differ in software management and release cycles.
